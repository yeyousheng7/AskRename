# AskRename

> An AI-powered batch renaming tool for everyone

## Features

- 🤖 AI-assisted smart renaming suggestions
- 📁 Batch file renaming with preview
- 🔄 Undo/Redo support
- 🖥️ Cross-platform (Windows, macOS, Linux)

## Tech Stack

- **Framework**: Electron + React + TypeScript
- **Build Tool**: electron-vite
- **Styling**: Tailwind CSS

## Getting Started

### Prerequisites

- Node.js >= 18
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
